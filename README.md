# 🛒 Flipkart Sales Analysis using Python

## 📌 Project Overview

This project analyzes a Flipkart sales dataset using Python to uncover business insights related to customer purchasing behavior, revenue generation, product performance, discounts, delivery efficiency, and sales trends.
The analysis includes data cleaning, feature engineering, exploratory data analysis (EDA), and data visualization to support business decision-making.

---

## 🎯 Business Objectives

- Identify top-performing product categories and brands.
- Analyze customer demand across different cities.
- Measure revenue generated by products and sellers.
- Understand the impact of discounts on sales.
- Evaluate delivery performance and customer ratings.
- Discover sales trends over time.
- Generate actionable business insights using data.

---

## 📂 Dataset

The dataset contains e-commerce sales information including:

- Product Name
- Category
- Brand
- Seller
- Final Price
- Discount Percentage
- Units Sold
- Revenue
- Customer Rating
- Review Count
- Payment Mode
- City
- Delivery Days
- Listing Date,etc...
